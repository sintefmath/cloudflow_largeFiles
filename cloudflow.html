CloudFlow upload large file

The CloudFlow version

Here we will try to find the look and feel of the web application for uploading large files to CSUC (for the first round at least).

File Upload widget with multiple file selection, drag&drop support, progress bar and validation for jQuery.
Supports cross-domain (not validated), chunked (not yet) and resumable (not yet) file uploads.
Should be used with any the PHP server sat up in the same project.

Workflow related instructions

This page should be accessed roughly like this in CloudFlow:
  • File Chooser is extended with "Upload Large File" context-menu, starting a Web Application with this part.
  • The user uploads files here. They will go into the folder chosen from the File Chooser.
  • When the user is happy, the "Finished uploading" button is pressed. The view should then go back to the FileChooser that started the Large File Uploader.
  • The user needs to refresh the folder to see the new files... FC can be "paused" by a pop up - need to press "Okay" to confirm upload done and refresh.
